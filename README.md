# Mohamed Najeh — Personal Portfolio Website

> Live at **[mohamednajeh.com](https://mohamednajeh.com)**

Personal portfolio website for **Mohamed Najeh**, Senior iOS Engineer with 5+ years of experience building high-performance apps in ride-hailing, healthcare, real estate & e-commerce. Based in Riyadh, available remotely worldwide.

---

## Features

- **Bilingual** — Arabic (RTL) & English, switchable at runtime
- **Dark mode** design with custom color system
- **Fully responsive** — mobile-first layout
- **SEO optimized** — Open Graph, Twitter Card, structured data (JSON-LD), canonical URL
- **Contact form** — powered by Formspree
- **Zero dependencies** — single HTML file, no build step required
- Deployed via **GitHub Pages** with a custom domain

## Local Development

No build tools needed. Just open `index.html` in your browser:

```bash
open index.html
```

Or use any local server:

```bash
npx serve .
# or
python3 -m http.server 8080
```

---

## Deployment

The site auto-deploys to GitHub Pages on every push to `main`.

**Custom domain setup:**
1. Add a `CNAME` file with `mohamednajeh.com`
2. Point DNS A records to GitHub Pages IPs at your registrar
3. Enable HTTPS in GitHub Pages settings
